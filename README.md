# Task 2: Customer Retention & Churn Analysis

## Project Overview

This project analyzes customer churn data to understand why customers leave and how customer retention can be improved.

## Tools Used

* Microsoft Excel
* PivotTables
* Excel Charts

## Key Metrics

* Total Customers: 7043
* Churned Customers: 1869
* Churn Rate: 26.54%

## Key Findings

* Month-to-month contract customers had the highest churn.
* Fiber optic customers had the highest churn count.
* Electronic check customers had higher churn.
* Customers on one-year and two-year contracts had lower churn.

## Recommendations

* Offer discounts for customers who choose longer contracts.
* Give retention offers to month-to-month customers.
* Improve Fiber optic pricing and customer support.
* Encourage automatic payment methods with small incentives.
